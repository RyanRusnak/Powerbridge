PowerBRIDGE Web application

PowerBRIDGE was built using ruby 1.8.7 and Rails 3.2.6 but is compatible with other versions of ruby as well and older versions of Rails.

Getting started:
Install ruby
Install rails
Open terminal
Navigate to project folder in terminal
type 'bundle install' to install the gems in the included gemfile
type 'rake db:migrate' to create the database
type 'rails server' to start the server. 
Launch your web browser
Type in 'localhost:3000/customers' to access the application

The styling of the web application is done with twitter bootstrap which uses LESS.
Styling can be overridden using LESS or CSS under the assets/stylesheets folder.
